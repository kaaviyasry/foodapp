# FoodApp UI/UX Design (SweetBite) 🍰

A modern and clean Food Ordering Mobile App UI designed using Figma.  
This project includes complete user flow from onboarding to payment confirmation.

## 📌 Screens Included
- Splash & Onboarding
- Sign Up / Login
- Home Screen
- Product Details
- Cart
- Payment
- Order Confirmation
- Location / Map Screen

## 🛠 Tools Used
- Figma
